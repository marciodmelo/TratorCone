# TratorCone — Jogo em PyGame

Projeto acadêmico desenvolvido em Python com a biblioteca PyGame, com o objetivo
de praticar Programação Orientada a Objetos, organização de código e lógica de jogos.

## Sobre o jogo
Jogo de corrida com obstáculos, com suporte a:
- Modo 1 jogador
- Modo 2 jogadores
- Sistema de pontuação
- Menu interativo
- Persistência de score

## Conceitos aplicados
- Programação Orientada a Objetos (POO)
- Game Loop
- Eventos do PyGame
- Detecção de colisão
- Organização modular do código
- Persistência de dados com SQLite

## Tecnologias
- Python 3
- PyGame
- SQLite

## Como executar
```bash
pip install -r requirements.txt
python main.py
